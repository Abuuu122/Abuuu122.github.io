Real-Time Global Illumination (in 3D)

Reflective Shadow Maps(RSM)​

如何快速计算一次bounce

光源直接照亮的patch可以由shadow map得到

每个patch都可以视作次级光源

不用考虑所有patch对p处的贡献，可以像PCSS一样只取周围一定范围内的patch，从而加快速度

Light Propagation Volumes (LPV)​

场景中用3D网格来传播radiance，数量比体素小一个数量级

通过RSM的方法生成直接照射到的radiance point

每个格子里的radiance point累加起来用SH表示为各个方向的radiance分布

相邻的六个格子迭代传播四五次直至稳定收敛

对每个shading point都能得到各个方向传播来的间接光照

problem:格子较大，背面可能被照亮

Voxel Global Illumination (VXGI)​

比LPV效果好，但是开销更大

场景描述为hierarchical voxels(有开销)

light pass记录光照和反射方向

camera pass渲染时shading point发出cones(数量由材质决定)采样

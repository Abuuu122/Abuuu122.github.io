Real-Time Shadows 2

More on PCF and PCSS​

CHI：阶跃函数，大于0为1，小于0为0，得到[0, 1]的Visibility

不是对shadow map滤波，也不是对图像滤波，而是对shadow map上一定范围内相对距离取（加权）平均得到Visibility，用到渲染方程中形成软阴影。

PCSS的第一步(取范围内blocker平均深度)和第三步(取范围内Visibility)会有大开销

Variance Soft Shadow Mapping（VCSS）​

是对PCSS的性能优化

对第三步的优化

求Visibility是求shadow map一定范围内有百分之多少的texel距离更近，需要采样范围内所有texel才能求出，采样还会造成noise，可以用范围查询替代。

Temporal denoising应用后采样造成的noise可以优化掉，现在还是PCSS用得更多。

**假设：**范围内的texel的深度为正态分布

如果知道Mean和Variance就能很快得到百分比

Var(X) = E(X^2) - E^2(X) => 除了深度的map还要另外保存一个深度平方的map

Mean：用MIPMAP或SAT

MIPMAP and Summed-Area Table(SAT)​

在VSSM中解决快速求Mean的问题

MIPMAP只能用于正方形，SAT适用与矩形

Moment shadow mapping​

VSSM的缺点：假设不成立时效果不好，真实分布与正态分布截然不同，造成本该产生阴影的地方出现light leaking

Moment Shadow Mapping使用更高阶的矩(moments)来表示分布，更好地拟合。

Distance field soft shadows

SDF为空间中每个点离最近的物体的距离，在几何体的融合上很自然

  • Usage 1：Ray marching

    从一点出发，根据SDF得到安全距离(没有障碍物)，可以快速移动

    （贴墙会比较慢？）

  • Usage 2： Soft Shadow

    在marching的同时，记录最小的角度，根据角度得到遮挡的百分比

    Pros：快，质量高

    Cons：需要预计算SDF，存储SDF

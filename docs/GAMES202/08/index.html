Real-Time Global Illumination(screen space)

Screen Space Ambient Occlusion (SSAO)​

an approximation of global illumination in screen space

direct light加上ambient，ambient基于遮挡形成的visibility

怎么求visibility？

表面的点取一个球的范围采样一系列点

根据depth map判断点是否被遮挡，超过半数被遮挡就考虑AO

第三个球为例：10个点，9个被遮挡，考虑AO，visibility=1/5

会有错误判断遮挡的情况，这是因为没有考虑法向

HBAO(Horizon based ambient occlusion)可以避免这种错误遮挡的情况

SSDO（Screen Space Directional Occlusion）

与SSAO相反，采样中被遮挡的部分贡献次级光照，没遮挡的部分不贡献

都只能算小范围内的全局光

算更大范围的全局光，可以用SSR

SSR（Screen Space Reflection）

类似于ray tracing，光线与屏幕空间的壳求交

求交时步长可以试探性地动态变化，屏幕空间深度的壳采用min pooling的mipmap记录，从最细的层开始，如果没有相交就用更粗的层增进步长，从而加速。

几何会有问题，因为只反射了屏幕空间能看到的部分，其余部分被cut了，丢失了屏幕空间以外的信息，在做diffuse时比较困难
